# Credit-Card-Fraud-Detection# 💳 Credit Card Fraud Detection

## 📌 Description
A machine learning project for detecting fraudulent credit card transactions using a highly imbalanced Kaggle dataset with PCA-based features (V1–V28). The model analyzes transaction patterns to classify transactions as legitimate or fraudulent.

## 📊 Dataset
- Source: Kaggle
- Features: V1–V28 (PCA transformed), Time, Amount
- Target: Class (0 = Normal, 1 = Fraud)

## 🧠 Approach
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Scaling
- Model Building
- Evaluation (Precision, Recall, ROC-AUC)

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook

## 🚀 How to Run
1. Clone repo
2. Open notebook
3. Run all cells
